# Template de Especificação — LeadFinder Pro

> Use este template ao criar SPEC.md ou issues/ para novas features.

## Formato SPEC.md

```markdown
# [Nome da Feature]

## Overview
[O que faz em 2-3 linhas]

## [/rota-da-pagina]

### Components
- **NomeEmPascalCase**: [responsabilidade]

### Behaviors
- **nome-em-kebab-case**: [o que acontece]
```

## Formato de Issue

```markdown
# Implement [nome-do-behavior] in [/rota]

[Descrição em 1-2 linhas]

## Functional Specification
### Preconditions
### Happy Path
### Edge Cases
### Error Cases

## Files to Create / Modify
## Tasks
- [ ] ...
```
